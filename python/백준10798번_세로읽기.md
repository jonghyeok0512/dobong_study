# 오늘 학습 요약
비정형 2차원 배열(행마다 길이가 다른 배열)을 세로 방향으로 순회하며 데이터를 추출하는 알고리즘을 학습함. 인덱스 참조 오류(IndexError)를 방지하기 위한 조건문 설정과 행(Row)/열(Column)의 올바른 인덱싱 방법을 익힘.

---

## 배운 개념 정리
### 1. 세로 읽기 (Column-wise Traversal)
- **정의**: 2차원 배열에서 가로줄(행)이 아닌 세로줄(열)을 기준으로 데이터를 읽는 방식.
- **중요성**: 데이터베이스의 특정 컬럼 데이터를 수집하거나, 행렬의 전치(Transpose) 연산의 기초가 됨.
- **주의점**: 모든 행의 길이가 같지 않을 경우, 존재하지 않는 인덱스를 참조하여 `IndexError`가 발생할 수 있음.

### 2. 인덱스 안전장치 (`if j < len(mat[i])`)
- 반복문에서 열 인덱스 `j`가 현재 행 `mat[i]`의 실제 길이보다 작을 때만 접근하도록 제한하여 프로그램이 강제 종료되는 것을 방지함.

---

## 오늘 작성한 코드/풀이 (최종본)

```python
# 1. 입력 데이터 구조화 (비정형 2차원 리스트)
mat = [
    ['A', 'A', 'B', 'C', 'D', 'D'],
    ['a', 'f', 'z', 'z'],
    ['0', '9', '1', '2', '1'],
    ['a', '8', 'E', 'W', 'g', '6'],
    ['P', '5', 'h', '3', 'k', 'x']
]

# 2. 세로 읽기 로직 구현
# 각 행의 최대 길이는 문제 기준 15이므로 range(15)를 써도 되지만, 
# 실전성을 위해 첫 번째 행의 길이를 기준으로 실습함.
for j in range(len(mat[0])): 
    for i in range(len(mat)):
        # [생각의 흐름]
        # i는 줄 번호(0~4), j는 칸 번호(0~5)
        # 모든 줄의 j번째 칸이 있는 것은 아니므로, 줄의 길이와 현재 칸 번호를 비교함.
        if j < len(mat[i]):
            # 주소는 항상 mat[행][열] 순서임을 잊지 말 것!
            print(mat[i][j], end="")

# 결과: Aa0aPAf985Bz1EhCz2W3D1gkD6x
```

---

## 실수/에러와 해결 과정
### ❌ 실수 사례
- **현상**: `IndexError: list index out of range` 발생 및 출력 값 꼬임.
- **원인 1**: `for j in range(len(mat[0]))` 사용 시, 다른 행들이 첫 번째 행보다 짧으면 참조 에러 발생.
- **원인 2**: 출력할 때 `mat[j][i]`라고 작성함. `j`가 바깥 루프 변수라고 해서 행 인덱스 자리에 넣으면, 행의 개수(5)보다 큰 `j`값이 들어올 때 에러가 남.

### ✅ 해결 과정
- **해결 1**: `if j < len(mat[i])` 조건문을 추가하여 해당 칸이 존재하는지 매번 확인하게 함.
- **해결 2**: 반복 순서가 바뀌어도 데이터의 고유 주소는 `mat[행][열]` 구조임을 인지하고 `mat[i][j]`로 수정함.

## 코드 리뷰 (성능 및 실무 관점)
### 1. 시간 복잡도
- $O(R \times C)$: 행의 개수($R$)와 최대 열의 개수($C$)만큼 순회하므로 효율적임.

### 2. 실무/RAG 관점의 참견
- **데이터 패딩(Padding)**: 실무에서는 이렇게 비어있는 칸을 무시하기보다, 공백(` `)이나 특수 기호(`PAD`)로 채워 넣어 행렬의 모양을 직사각형으로 맞춘 뒤 처리하는 경우가 많습니다. 그래야 나중에 AI 모델(TensorFlow, PyTorch 등)에 데이터를 한 번에 집어넣을 수 있기 때문입니다.

## 이번 학습으로 얻은 내용
- 2차원 배열에서 인덱스 `i`와 `j`가 각각 '세로 위치'와 '가로 위치'를 결정한다는 사실을 명확히 이해함.
- 에러 메시지를 통해 내가 짠 코드의 논리적 허점(주소 참조 오류)을 스스로 찾아내는 훈련을 함.

## 백준 제출용
```python
import sys

# 1. 5줄의 입력을 받아서 2차원 리스트로 만들기
mat = []
for _ in range(5):
    # 각 줄의 공백을 제거하고 리스트로 변환해서 추가
    line = sys.stdin.readline().strip()
    mat.append(list(line))

# 2. 세로로 읽기 (최대 15글자 기준)
for j in range(15): # 열(칸)
    for i in range(5): # 행(줄)
        # 현재 줄(i)에 j번째 글자가 있는지 확인
        if j < len(mat[i]):
            print(mat[i][j], end="")
```
