#  369 게임 문제 풀이 정리 (BEGINNER)

##  문제 개요
369 게임은 1부터 n까지 숫자를 순서대로 확인하면서  
숫자 안에 **3, 6, 9 중 하나라도 포함되어 있으면 손뼉을 치는 게임**이다.

이번 문제에서는 두 가지 규칙 버전이 있었다.

---

##  369게임 1 (손뼉은 최대 1번)

###  규칙
- 숫자에 **3, 6, 9 중 하나라도 포함되면**
- **무조건 손뼉 1번**
- 중복 등장해도 횟수는 1번만 증가

###  핵심 아이디어
- 숫자를 **문자열로 변환**
- `'3' in 문자열` 같은 방식으로 포함 여부만 체크
- OR 조건 사용

###  사용한 핵심 문법
- `str(num)`
- `in`
- `or`
- `range()`
- `for문`

###  코드
```python
def solution(n):
    answer = 0

    for num in range(1, n + 1):
        str_n = str(num)
        if '3' in str_n or '6' in str_n or '9' in str_n:
            answer += 1

    return answer
```

###  배운 점
- **“포함 여부만 확인할 땐 in 연산자”**가 가장 직관적
- 문자열 변환은 숫자 자릿수 문제에서 거의 필수 스킬

---

##  369게임 2 (포함된 개수만큼 손뼉)

###  규칙
- 숫자에 포함된 **3, 6, 9의 개수만큼 손뼉**
- 예)
  - 13 → 1번
  - 36 → 2번
  - 999 → 3번

###  핵심 아이디어
- 숫자를 문자열로 변환
- **각 자리 하나씩 검사**
- 조건을 만족할 때마다 `answer += 1`

###  사용한 핵심 문법
- 문자열 순회 (`for char in str`)
- `in` + `set`
- 중첩 for문

###  코드
```python
def solution(n):
    answer = 0

    for num in range(1, n + 1):
        str_n = str(num)
        for char in str_n:
            if char in {'3', '6', '9'}:
                answer += 1

    return answer
```

###  배운 점
- **“포함 여부” vs “포함 개수”는 로직이 완전히 다르다**
- `if '3' in str` → 존재 확인
- `for char in str` → 개수 계산
- `set`을 사용하면 조건이 더 깔끔해진다

---

##  두 문제 비교 정리

| 구분 | 369게임 1 | 369게임 2 |
|----|----|----|
| 손뼉 횟수 | 최대 1번 | 포함된 개수만큼 |
| 핵심 로직 | 포함 여부 체크 | 문자 하나씩 카운트 |
| 핵심 문법 | `in`, `or` | 중첩 for문 |
| 난이도 포인트 | 조건식 이해 | 자릿수 분해 사고 |

---

##  실무적으로 중요한 포인트
- 숫자 문제 → **문자열로 바꾸는 사고**
- 조건이 단순하면 **논리 연산자**
- 개수 계산 문제는 **반복 + 누적**
- 문제 설명을 정확히 읽지 않으면 로직이 완전히 달라진다
